# 🏛️ Historaunt - A Historical Restaurant Website

## Overview
Historaunt is a unique restaurant website concept where diners can experience meals from different historical periods and famous minds throughout history.

## Features

✨ **Historic Theme**
- 1800s Classics section
- 1900s Era Meals
- Famous Minds Menu (inspired by great historical figures)
- Desserts & Drinks

📱 **Responsive Design**
- Works seamlessly on desktop, tablet, and mobile devices
- Sticky navigation for easy browsing

🎯 **Interactive Elements**
- Smooth scrolling navigation
- Online reservation form
- Hover effects on menu items
- Scroll animations

📍 **Contact & Information**
- Location and hours
- Phone and email contact
- Reservation system

## File Structure

```
├── index.html      # Main HTML file with content
├── styles.css      # All styling and responsive design

```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Explore the menu and make a reservation!

## Customization

### Change Colors
Edit `styles.css` and modify the color values:
- Primary color: `#3b2f2f` (dark brown)
- Secondary color: `#6b4f4f` (lighter brown)
- Accent color: `#f9d5a4` (gold)

### Update Menu Items
Edit the menu sections in `index.html` by modifying the `.item` divs:
```html
<div class="item"><span>Dish Name</span><span>$Price</span></div>
```

### Add More Sections
Add new menu categories by copying:
```html
<h2>🎨 Your Section Title</h2>
<div class="item"><span>Item</span><span>$Price</span></div>
```
